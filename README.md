# Nexdigm Annual Increment Processing Application - Documentation

Documentation and project-management repo for the Nexdigm Annual Increment Processing Application - a controlled web application, built by BetaCraft (Soni Technologies LLC) for Nexdigm HR & Finance, that replaces the spreadsheet-driven annual compensation exercise for ~1,500 employees across four countries.

See CLAUDE.md for full context and session rules, OWNERS.md for write zones, and shared/DECISIONS.md for the single source of truth on locked decisions. The confirmed requirements understanding lives in architecture/specs/BRD_nexdigm-hr-tool.md (with a companion summary).
